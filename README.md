# Cosmo Website Framework V1 (derived from Luna Landing)

A template derived from the [Luna Landing](https://github.com/JimmyCamus/luna-landing) template. This repo builds upon this template with added sections and greater customizability. An innovative and highly customizable landing page that allows you to create and modify the website easily and quickly through a JSON file. Harnessing the power of Astro, a modern and efficient web development tool, Luna Landing provides you with an unparalleled development experience, enabling you to focus on your site's design and content without worrying about the complexity of code.

## Key Features

- Extreme customization & simplfied maintenance via JSON file: Easily change colors, fonts, images, and text without the need for manual code editing.

- Fast load times and performance thanks to AstroJS

- Mobile-first development

## 🛠️ Stack

- Astro - The web framework for content-driven websites.
- Typescript - JavaScript with syntax for types.
- Tailwind CSS - A utility-first CSS framework for rapidly building custom designs.
- DaisyUI - A Tailwind framework featuring pre-made components
